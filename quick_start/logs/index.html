    Logs¶

    There is loguru library under the hood.

    Settings¶

    import sys

from loguru import logger

logger.add(
    sys.stderr,
    format="{time} {level} {message}",
    filter="my_module",
    level="INFO",
)

    Disable/Enable¶

    from loguru import logger

logger.disable('extract_emails')
logger.enable('extract_emails')
